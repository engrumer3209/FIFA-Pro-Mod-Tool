**Does this tool change my game files permanently?**
The tool creates a backup of your original game files. You can restore the original state at any time by using the "Restore Backup" option in the Settings menu.

**Can I use this for online play?**
Do not use mods while connected to official online game servers. Use this tool for offline modes only to prevent account bans.

**What file formats does the tool support?**
You can import common image formats like PNG and DDS for textures. Database files use standard table formats accessible through the app interface. If you encounter an error, check that your file size matches the original asset dimensions.
